Jupyter Notebook은 웹 기반 인터랙티브 컴퓨팅 환경임. 쉽게 말해서 웹 브라우저에서 코딩하고, 결과 바로 확인하고, 텍스트도 쓰고, 이미지도 넣고, 심지어 수식까지 넣을 수 있는 개꿀 에디터임. 데이터 분석, 머신러닝, 교육 자료 만들 때 진짜 많이 씀.

**핵심 기능:**

*   **코드 실행:** Python, R, Julia 등 다양한 언어 지원함. 커널만 설치하면 됨. 코드 셀에 코드 작성하고 실행하면 결과 바로 보여줌.
*   **Markdown 지원:** 코드 설명이나 문서화할 때 Markdown 문법 쌉가능. 텍스트 서식, 링크, 이미지 삽입 등등.
*   **인라인 결과:** 코드 실행 결과 (표, 그래프, 이미지 등) 코 바로 아래에 보여줌.
*   **대화형 인터페이스:** 코드 수정하고 다시 실행하는 거 개편함. 실험하고 결과 바로 보는 거에 최적화됨.
*   **노트북 저장:** `.ipynb` 확장자로 저장됨. 코드, 텍스트, 결과 다 같이 저장되니까 공유하기도 좋음.

**장점:**

*   **접근성:** 웹 브라우저만 있으면 어디서든 사용 가능.
*   **편의성:** 코드 작성, 실행, 결과 확인 한 번에.
*   **시각화:** Matplotlib, Seaborn 같은 라이브러리 연동해서 그래프 쉽게 그림.
*   **공유:** `.ipynb` 파일로 공유하거나 HTML, PDF 등으로 export 가능.
*   **학습:** 코딩 배우는 데 개좋음. 코드랑 설명 같이 보면서 따라하기 쉬움.

**단점:**

*   **웹 기반:** 인터넷 연결 안 되면 좀 불편함.
*   **협업:** 여러 명이 동시에 편집하기는 좀 빡셈. (JupyterLab 같은 확장 기능 쓰면 좀 나아짐)
*   **보안:** 웹 기반이라 보안에 좀 신경 써야 함.

**사용법:**

1.  **설치:** Anaconda 같은 배포판 설치하면 Jupyter Notebook 기본으로 깔림. 아니면 `pip install jupyter` 로 설치 ㄱㄱ.
2.  **실행:** 터미널에서 `jupyter notebook` 입력하면 웹 브라우저에서 실행됨.
3.  **새 노트북:** "New" 버튼 누르고 원하는 언어 선택.
4.  **코드 작성:** 코드 셀에 코드 쓰고 `Shift + Enter` 눌러서 실행.
5.  **Markdown 작성:** Markdown 셀 만들어서 텍스트 작성.

**꿀팁:**

*   JupyterLab 쓰면 더 많은 기능 활용 가능. (파일 관리, 편집 기능 강화 등)
*   Magic commands 활용하면 코드 실행 시간 측정, 외부 파일 로딩 등 다양한 작업 가능.
*   Kernel 관리 잘해야 함. 필요 없는 Kernel 삭제하고, 필요한 Kernel 설치하고.

**결론:**

Jupyter Notebook은 데이터 분석, 머신러닝 하는 사람들한테 완전 필수템임. 코딩 초보도 쉽게 사용할 수 있고, 결과 바로바로 확인하면서 실험하기 좋음. 한 번 써보면 왜 다들 Jupyter Notebook 쓰는지 알게 될 거임.
