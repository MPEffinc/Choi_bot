**1차원 가우시안 확률밀도함수 (Gaussian Probability Density Function, PDF)가 뭔데?**

가우시안 분포는 정규 분포라고도 하는데, 통계학에서 젤 중요한 분포 중 하나임. 자연 현상이나 사회 현상 데이터 분석할 때 진짜 많이 쓰여. 1차원 가우시안 PDF는 쉽게 말해서 종 모양 곡선!

**수식**

수식은 다음과 같음:

```
f(x) = (1 / (σ * sqrt(2π))) * exp(-(x - μ)^2 / (2σ^2))
```

여기서:

*   `x`: 확률 변수 (니가 측정하는 값)
*   `μ`: 평균 (mean). 종 모양 그래프의 중심 위치. 평균이 바뀌면 그래프가 좌우로 움직여.
*   `σ`: 표준 편차 (standard deviation). 데이터가 평균으로부터 얼마나 흩어져 있는지 나타내는 값. 표준 편차가 크면 그래프가 넓적해지고, 작으면 뾰족해짐.
*   `π`: 원주율 (대략 3.14159)
*   `exp`: 지수 함수 (e^x)

**개형 (Shape)**

1.  **종 모양 (Bell Shape):** 젤 중요한 특징! 가운데가 볼록하고 양쪽 꼬리가 점근선처럼 x축에 가까워짐.

2.  **대칭 (Symmetry):** 평균 μ를 기준으로 완벽하게 대칭임. 즉, 평균을 중심으로 왼쪽이랑 오른쪽 모양이 똑같다는 뜻.

3.  **최댓값 (Maximum):** x = μ 일 때 최댓값을 가짐. 이 최댓값은 `1 / (σ * sqrt(2π))` 임.

4.  **점근성 (Asymptotic):** x가 ±∞로 갈 때, 함수 값은 0에 가까워짐. 즉, x축에 닿지는 않지만 계속 가까워지는 형태임.

5.  **변곡점 (Inflection Points):** x = μ ± σ 에서 변곡점이 생김. 변곡점은 그래프의 오목함수가 바뀌는 지점을 말함.

**Dirac 표기법 (Dirac notation)**

Dirac 표기법은 양자역학에서 상태 벡터와 연산자를 표현하는 표준적인 방법임. Paul Dirac이라는 물리학자가 개발했음. "Bra-ket" 표기법이라고도 함. 복잡한 수식을 간결하게 표현하고 물리적 의미를 명확하게 드러내는 데 유용함.

**기본 구성 요소**

*   **Ket 벡터 (|⟩):** 힐베르트 공간의 상태 벡터를 나타냄. 예를 들어, |ψ⟩는 상태 ψ를 나타내는 ket 벡터임. 열벡터(column vector)로 생각하면 됨.
*   **Bra 벡터 (⟨|):** Ket 벡터의 쌍대 벡터를 나타냄. Ket 벡터 |ψ⟩에 대응하는 bra 벡터는 ⟨ψ|임. 행벡터(row vector)로 생각하면 됨. 복소수 켤레(complex conjugate) 연산을 포함함.
*   **Bra-Ket (⟨|⟩):** Bra 벡터와 Ket 벡터의 내적을 나타냄. ⟨φ|ψ⟩는 상태 φ와 상태 ψ의 내적 (스칼라곱)을 의미함. 파동함수의 겹침 정도나 확률 진폭을 계산할 때 사용됨.

**표기법 규칙**

*   **Ket 벡터:** |⟩ 기호 안에 상태를 나타내는 기호 (ψ, φ, n 등)를 넣음.
*   **Bra 벡터:** ⟨| 기호 안에 해당 Ket 벡터의 상태 기호를 넣음. 순서가 중요함.
*   **선형성:** a와 b가 복소수이고 |ψ⟩와 |φ⟩가 Ket 벡터일 때, a|ψ⟩ + b|φ⟩는 또 다른 Ket 벡터임. Bra 벡터도 마찬가지임.
*   **정규직교성:** 서로 다른 상태 벡터 |ψ⟩와 |φ⟩가 직교하면 ⟨φ|ψ⟩ = 0임. 같은 상태 벡터의 내적 ⟨ψ|ψ⟩ = 1이면 정규화되었다고 함.
*   **완전성 관계:** 힐베르트 공간의 모든 상태 벡터는 기저 벡터들의 선형 결합으로 표현 가능함. {|n⟩}이 완전한 정규직교 기저이면, ∑|n⟩⟨n| = I (단위 연산자)가 성립함.

*   **스핀 표현:** |↑⟩는 스핀-업 상태, |↓⟩는 스핀-다운 상태를 나타내는 Ket 벡터임.

**Dirac 표기법의 장점**

*   **간결성:** 복잡한 수식을 짧고 명확하게 표현 가능.
*   **물리적 의미 강조:** 상태 벡터와 연산자의 물리적 의미를 직관적으로 파악 가능.
